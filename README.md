# hiproxy-plugin-devtools

 hiproxy插件，实时打印hiproxy代理请求的信息。

 ![](https://github.com/hiproxy/hiproxy-plugin-devtools/blob/master/devtools.gif)
 
 
## 安装

 npm i hiproxy-plugin-devtools -g
 
 
## 使用

 启动hiproxy, 访问页面 http://127.0.0.1:5525/devtools
